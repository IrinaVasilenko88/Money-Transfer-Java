**Для запуска проекта:**
1. Склонировать проект из репозитория командой 

```
git clone https://github.com/IrinaVasilenko88/Money-Transfer-Java.git
``` 
2. Открыть склонированный проект в Intellij IDEA
3. Запустить команду ```run```
